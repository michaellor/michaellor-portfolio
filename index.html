Shoe Store Project

This project allows for the addition of Shoe Stores and Shoe Brands. A database is used to create, store, and manage data for the Store class, the Brand class, and a join class between the two. Along with adding stores and brands, stores can also be deleted or updated with a new name. The join class allows for many to many relationships, which ties data between both classes, thus allowing the cross referencing of data between the two classes.

This project uses PHPUnit for unit testing. The project uses PHP, Twig, Silex, MySQL, and Composer.
